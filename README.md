## 프로젝트 기본 정보
- 본 프로젝트는 MYP Design Class 에서 제작한 대한민국 투어를 더욱 쉽고 빠르게 계획할 수 있도록 도와주는 사이트입니다.
- 본 프로젝트는 Bootstrap 5를 기반으로 제작된 프로젝트입니다.
- 본 프로젝트에 있는 온라인 스토어 페이지는 아무것도 되지 않는 깡통 웹사이트입니다. (로그인은 작동됨. 결재가 안된다는 뜻임)
